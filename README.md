# Automotive Sales Analysis — US Vehicle Market (1984–2015)

## Project Overview
Analysis of 531,128 US vehicle sales records to uncover trends in brand performance, pricing, geographic demand, and market resilience using SQL and data visualization.

## Tools Used
- Google BigQuery (SQL)
- Google Sheets (Charts & Visualization)
- Kaggle (Dataset)

## Repository Structure
- `/sql` — All BigQuery SQL queries
- `/charts` — Data visualizations
- `Insights_Report.pdf` — Full analysis report

## Key Findings
- **Ford** leads with 93,533 sales 55% more than Chevrolet
- **2008 Financial Crisis** caused a 34% sales drop, followed by 400% recovery by 2012
- **Sedans + SUVs** account for 70% of total market demand
- **Vehicle condition** is the strongest price predictor 4x price difference between condition 1 and 5
- **Florida** surpasses California in total sales despite smaller population

[Vehicle Sales and Market Trends Dataset — Kaggle](https://www.kaggle.com/datasets/syedanwarafridi/vehicle-sales-data)
